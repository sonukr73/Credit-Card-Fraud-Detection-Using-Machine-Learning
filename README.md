# Credit Card Fraud Detection using Machine Learning

This project detects fraudulent credit card transactions using machine learning
techniques on highly imbalanced datasets.

## Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn

## Workflow
- Data preprocessing
- Handling class imbalance
- Model training
- Model evaluation

## Models Used
- Logistic Regression
- Random Forest
- Isolation Forest

## Evaluation Metrics
- Precision
- Recall
- F1-score
- ROC-AUC
